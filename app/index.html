Running & Programming Environment

candle.js is a Game Engine that run in modern browser.candle.js is written entirely with JavaScirpt. Download Modern Browsers Chrome Safari Firefox

Build candle.js

candle.js is built with node.js, npm, yeoman, gulp, bower.

Install Node.js

Node.js is Javascript commandline environment. You need to install node first then check the version.

$ node -v
v0.12.0

npm is node package manager. we use it to install yeoman. -g means install yeoman globally.

Install Yeoman

$npm install -g yo

Yeoman Doctor

yeoman is a application to create application template. It does basic scaffolding for us.

Install Bower

$ npm install -g bower

bower download and install client side javascirpt for us.

Install Gulp

$ npm install -g gulp

gulp is a task manager, it build will build and complie the project.

Install gulp webapp generator

$ npm install -g generator-gulp-webapp

This is one of yeoman's generator

Create candlejs project

$ mkdir candlejs
$ cd candlejs
$ yo gulp-webapp

Now everything is generated for us and we can now launch gulp web server.

Gulp Server

$ gulp serve
 -------------------------------------
       Local: http://localhost:9000
    External: http://192.168.1.65:9000
 -------------------------------------

you can now access candle.js project with this url.

Git Version Control

$ git init
$ git commit -m 'init'
$ git remote add origin git@github.com:warycat/candlejs.git
$ git push -u origin master

git is a version control system used by candle.js project. Install git here. now the project source code is on github.

Deploy project on gh-pages

$ gulp
$ git add dist
$ git commit -m "Initial dist subtree commit"
$ git subtree push --prefix dist origin gh-pages

1st remove the dist directory from the project’s .gitignore file (it’s ignored by default by Yeoman). 2nd build project with gulp. gulp will compile your project into dist folder. 3rd commit your dist folder. 4th push dist subtree of your project to gh-pages. Now the project can be access on internet by the url http://warycat.com/candlejs
